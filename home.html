HTML Interview Questions

1. What is HTML5?

HTML5 is the latest version of HyperText Markup Language used to structure and present content on the web. It adds new elements, attributes, behaviors, and supports multimedia without needing extra plugins like Flash.

2.What is the difference between HTML and HTML5?

HTML is the standard markup language for creating web pages.
HTML5 is the latest version with new semantic tags, multimedia support, and APIs like localStorage and Geolocation.
HTML5 improves browser compatibility and reduces reliance on external plugins.

3. What are the new features in HTML5?

HTML5 includes semantic tags, multimedia elements, and offline storage.
It adds new input types and JavaScript APIs like Geolocation.
It improves accessibility, performance, and device compatibility.

4. What are semantic elements in HTML5?

Semantic elements clearly define the content's role. Examples include <article >,<section > , <nav >, and <footer >. They improve SEO and make code easier to understand.

6.What is the purpose of the tag in HTML?

The <meta> tag provides metadata about the HTML document, such as character encoding, page description, keywords, and viewport settings. It is placed inside the <head> section and helps with SEO and responsive design. For example: <meta name="viewport" content="width=device-width, initial-scale=1.0"> makes the page mobile-friendly.
